# 👋 About Me

Snowflake Certified Data Engineer specialising in building and evolving scalable cloud native data platforms using AWS, Snowflake, dbt, Python, and SQL.

I apply strong data modelling principles such as star schema design, incremental processing, and SCD Type 2 alongside modern data engineering practices including CI/CD automation, IaC using Terraform, automated testing with dbt and Pytest, containerisation using Docker, and secure IAM role design with least privilege access patterns.

Recently extended into LLM platform engineering, building out an internal LiteLLM gateway with end-to-end testing and observability.

🎯 Open to roles in the UK and remote.
